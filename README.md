# Web-imageEditor

A browser-based image editor built using HTML, CSS, JavaScript, and the Canvas API.

## Features

- Upload images
- Adjust brightness, contrast, saturation, blur, grayscale, sepia, opacity, and invert
- Apply preset filters
- Reset image settings
- Download edited images
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API

## Future Improvements

- Crop tool
- Rotate and flip image
- Before/After comparison
- Theme switcher
